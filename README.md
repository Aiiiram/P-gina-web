El primer bloque del código corresponde al reloj digital. Para hacerlo funcionar, utilicé la función iniciarReloj(), que obtiene la hora actual mediante un objeto Date. A partir de ahí extraigo las horas, minutos y segundos, y les añado un cero delante cuando son menores de diez para mantener el formato de reloj digital. Después actualizo el contenido del elemento con id reloj-digital y vuelvo a llamar a la función cada segundo usando setTimeout, de forma que el reloj se actualiza en tiempo real sin necesidad de recargar la página. 

También programé un sistema para ocultar y mostrar todas las imágenes de la página. Para ello declaré una variable llamada estado que actúa como interruptor. La función ocultar() selecciona todas las etiquetas <img> y, dependiendo del valor de estado, cambia su visibilidad entre “hidden” y “visible”. Cada vez que se ejecuta, el estado se invierte, lo que permite alternar entre mostrar y ocultar las imágenes con un solo botón. 

Luego está el contador, la función cargarContador() recupera el valor guardado en localStorage y lo muestra en pantalla, o lo inicializa a cero si no existe. La función cambiarContador() permite incrementar, decrementar o reiniciar el contador según la acción recibida, y siempre guarda el nuevo valor en localStorage para que se mantenga incluso al cerrar la página. Además, añadí la función actualizarColorContador(), que cambia el color del número según sea positivo, negativo o cero, siguiendo las indicaciones del ejercicio. 

La función evaluarAccesoParque(), que recoge los datos introducidos en un formulario: la edad de la persona, la edad del acompañante y si se ha marcado la casilla de autorización especial. A partir de esos valores aplico lo siguiente: los menores de 5 años no pueden entrar; los niños entre 5 y 12 necesitan un acompañante mayor de 18; las personas entre 12 y 60 entran sin restricciones; y los mayores de 60 solo pueden acceder si han marcado la autorización. Si la edad introducida no es válida, se muestra un mensaje de aviso.
